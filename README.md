**V2Ray客户端**

电脑 https://github.com/2dust/v2rayN/releases

使用:
- 点击左上角 订阅-订阅设置
- 点击左下角 添加
- 地址填上面其中一个链接，然后确定 *(备注可以不填)*
- 点击左上角 订阅-订阅更新 即可
- 在下边的工具栏右键V2rayN图标，点击Http代理-开启Http代理即可
- 切换路线，点击要用的路线后按Enter键即可

或

`Ctrl + C`复制链接，在软件里直接粘贴`Ctrl + V`，点击左上角 订阅-订阅更新 即可

手机 https://github.com/2dust/v2rayNG/releases
使用:
- 复制链接
- 点击 右上角+ 从剪贴板导入
- 点击 右上角: 更新订阅 即可
- 点击想用的路线，再点击右下角的大 V 即可启用VPN
